This quantum simulation for the Thesis:

Quantum Artificial Life: simulating Lotka-Volterra Equation

needs to have installed:

- Python: used version was 3.7.7
- qiskit: used version was 0.14.1
- Matlab API engine for Python

File 'quantumsimulation.ipynb' contains all functions defined, except the other two matlab scripts included.
Execution and plots are the two last cells
